1. Сборка образов

    docker-compose build

2. Запуск контейнеров

    docker-compose up

3. Запуск контейнеров в фоне

    docker-compose up -d

4. Сборка образа и запуск в фоне после успешной сборки

     docker-compose up -d —build

5. Выполнение команды внутри контейнера
    docker-compose exec
